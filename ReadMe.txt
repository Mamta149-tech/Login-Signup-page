Tools and Technology used:
Editor: VsCode
FrontEnd: HMTL, CSS, JavaScript, jQuery, Bootstrap
Backend: PHP, XAMPP
Database: MySQL

##################################################################################################

Working:
First of all open the index.php page which is the Registration page. On successful registration it redirects to Login.php which on successful login redirects to Thankyou.html
Also there is a link on Registartion page that will take you to Login page. Similarly, there is a link on Login page that will redirect you to Registration page.
All the uploaded profile pictures are stored in 'uploads' folder.

#####################################################################################################

Registration Form Validations:
->The First name and last name field can take only alphabets.
->The mother's name and father's name field can take only alphabets and spaces.
->The pincode field can take only 8-digits numbers.
->For a valid password there should be at least an uppercase letter, a lowercase letter and a number. The password should be more than or equal to 8.
->Confirm password field should have same value as password.
->There is only one registration possible with an email id. Email Id should be unique.
->There is only one registration possible with a profile picture. Profile picture should be unique.
->All the fields of the registration form are required.
